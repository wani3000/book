# 2026-07-22 전자책 상품 동기화 진단

## 목적

출시 후보 PDF 3권과 웹사이트 서재 파일, 각 상품 상세 페이지의 판매 문구가 같은 판본을 가리키는지 확인했다.

## 진단 결과

- 1권은 원본과 웹 배포본의 SHA-256이 일치하고, 230쪽·50단계 실전·경험편 24장이라는 상세 페이지 설명과 일치한다.
- 2권은 원본과 웹 배포본의 SHA-256이 일치하고, 90쪽·20개 장·통합 별첨이라는 상세 페이지 설명과 일치한다.
- 3권은 본문이 최신 48쪽 판본이었지만 PDF 첫 표지가 웹사이트의 최신 일러스트 표지가 아니었다. 본문은 유지하고 첫 표지만 최신 표지로 교체했다.
- 3권 상세 페이지의 오래된 78쪽 표기는 48쪽으로, 구성은 22개 장 + 통합 실습팩으로 바로잡았다.
- 공개 PDF에서 `박철완`, `서나라` 실명은 발견되지 않았다. 3권에서는 이전 직장 실명도 발견되지 않았다.

## 판단

원고를 새로 확장할 필요는 없으며, 이번 작업의 핵심은 최신 판본·표지·판매 메타데이터의 일치였다.
