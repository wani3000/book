# PHILIP BOOKS 회원 생애주기 운영·QA 문서

최종 갱신: 2026-07-21 (KST)

## 1. 적용 범위

Google 회원가입·로그인, 필수/선택 동의, 프로필 변경, 구매내역과 PDF 권한, 환불 상태, 로그아웃, 회원 탈퇴와 재가입까지를 한 흐름으로 관리한다.

## 2. 회원 상태와 동작

| 상태 | 로그인 | 마이페이지 | PDF | 운영 원칙 |
| --- | --- | --- | --- | --- |
| 신규 | Google 인증 후 필수 약관 동의 | 가입 완료 후 가능 | 구매 완료 후 가능 | 이용약관·개인정보 동의 일시와 버전 기록 |
| 활성 `active` | 즉시 가능 | 가능 | 유효한 `paid` 주문만 가능 | 마케팅 동의는 선택이며 언제든 변경 가능 |
| 정지 `suspended` | 불가 | 불가 | 불가 | 관리자 문의 안내 |
| 탈퇴 `deleted` | 자동 복구 금지 | 불가 | 불가 | 재가입 안내와 필수 동의를 다시 표시 |
| 재가입 | Google 재인증과 필수 동의 후 가능 | 기존 거래 기록 연결 | 유효한 구매 권한 복원 | 재가입 일시와 최신 동의 버전 기록 |

## 3. 구현된 안전장치

- 신규 회원은 Google 인증만으로 가입되지 않는다. 이용약관과 개인정보 수집·이용 동의를 확인해야 한다.
- 선택 마케팅 동의는 필수 동의와 분리하며 미동의여도 기능 제한이 없다.
- 탈퇴한 계정은 다시 로그인해도 자동 활성화되지 않는다.
- 재가입 화면에서 기존 구매내역과 유효한 PDF 권한이 복원된다는 사실을 먼저 알린다.
- 탈퇴는 `회원 탈퇴` 문구 직접 입력과 확인 체크를 모두 요구한다.
- 환불 신청이 `requested` 또는 `reviewing`이면 탈퇴를 막아 환불 결과를 받을 권리를 보호한다.
- 탈퇴 즉시 세션 쿠키를 폐기하고 회원 상태를 `deleted`로 변경하므로 서버 PDF 권한도 중단된다.
- 주문·결제·환불 기록은 법정 보존을 위해 유지하며 공개 프로필·마케팅 동의는 비활성화한다.
- 구매내역에는 상품, 주문번호, 결제수단, 금액, 구매일, 최초 열람과 환불 상태를 표시한다.

## 6. 개인정보·거래기록 원칙

회원 프로필은 탈퇴 시 비식별 상태로 전환한다. 주문·결제·환불 기록은 전자상거래 법정 보존과 분쟁 대응에 필요한 기간 동안 유지한다. 재가입은 이용자의 명시적 선택으로만 수행하며, 최신 이용약관·개인정보 동의 버전을 다시 기록한다.
