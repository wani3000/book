# 상세페이지 고정 구매 카드 스크롤 QA

검수일: 2026-07-23  
검수 화면: 데스크톱 1440×900, `/codex`

## 결과

1. 초기 스크롤 상태 — 주의
   - 구매 카드는 정상적으로 보이지만, 더 내려가 고정 상태가 되기 전이라 겹침 여부를 확정할 수 없는 상태였다.
   - 증거: `01-before-scroll.jpg`

2. 고정 상태 — 문제 확인
   - 상세 탭은 화면 상단 72–131px을 차지하지만 구매 카드는 76px부터 고정되어 상단 55px가 탭 아래로 가려졌다.
   - 판매가와 가격 일부가 보이지 않아 구매 정보의 위계와 가독성이 훼손됐다.
   - 증거: `02-before-clipped.jpg`

3. 수정 후 고정 상태 — 정상
   - 구매 카드를 147px부터 고정해 상세 탭 아래 16px 여백을 확보했다.
   - 결제수단 심사 완료를 기다리라는 안내 문구와 관련 래퍼·스타일을 제거했다.
   - 비활성 `간편결제 준비 중` 버튼만 남고 판매가·가격·버튼이 모두 잘리지 않는다.
   - 증거: `03-after-fixed.jpg`

## 접근성 확인

- 비활성 버튼의 `disabled` 속성과 `간편결제 준비 중` 접근 가능한 이름을 유지했다.
- 고정 카드와 상단 탐색 탭의 시각적 겹침을 제거했다.
- 스크린샷 기반 확인이므로 키보드·스크린리더의 전체 구매 여정은 이번 범위에 포함하지 않았다.
